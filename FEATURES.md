# 🧶 Crochelle AI Features


# 🤖 AI Pattern Generator

## Purpose

Generate custom amigurumi crochet patterns from a simple text description.

Example Prompt:

> "Create a small pink dinosaur wearing a yellow hat."

The AI generates:

* 🧶 Materials List
* 🪝 Hook Size Recommendation
* 🧵 Yarn Recommendation
* 📏 Finished Dimensions
* ⭐ Difficulty Level
* ⏱ Estimated Time
* 📋 Step-by-Step Crochet Pattern


# 📚 Learning Paths

## Purpose

Provide structured learning journeys for crocheters of every skill level.

Instead of searching randomly online, users follow guided learning paths.

### Example Paths

* 🌱 Complete Beginner
* 🧸 Amigurumi Beginner
* 🎨 Pattern Designer
* 💼 Crochet Business Owner

Each path contains:

* Weekly Lessons
* Recommended Projects
* Practice Challenges
* Skill Milestones
* Progress Tracking


# 🎥 Curated Learning Library

## Purpose

Recommend the best existing crochet resources instead of recreating tutorials.

Resources may include:

* YouTube Videos
* Playlists
* Articles
* PDF Guides

Resources are selected based on:

* User Skill Level
* Current Learning Path
* Project Goals


# 🌳 Skill Tracker

## Purpose

Track every crochet skill the user has learned.

The AI will use this information to:

* Recommend appropriate projects
* Avoid generating patterns using stitches the user hasn't learned
* Suggest the next skill to learn
