# 🚀 Crochèlle MVP (Version 1)

## Goal

Build the first working version of Crochèlle that allows users to generate beginner-friendly crochet patterns.

---

# Core Features

## ✅ AI Pattern Generator

Users can describe a crochet plushie in natural language.

Example:

> "A small pink dinosaur wearing a yellow hat."

The app generates:

* Materials
* Yarn recommendation
* Hook size
* Difficulty
* Estimated time
* Step-by-step crochet pattern

---

## ✅ Save Pattern

Users can save generated patterns for later.

---

## ✅ Pattern History

Users can view previously generated patterns.

# Success Criteria

Version 1 is successful if:

* A user can generate a crochet pattern.
* The generated pattern is readable.
* The interface feels cozy and welcoming.
* Users want to come back and generate another project.
